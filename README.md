# node-typescript
The "perfect" starter template for node + typescript project

### Usage

### Installation

```bash
# Click Use This Template

# Install npm packages
npm install
```

### npm commands
    
#### Run
- `npm run dev` *auto reloads on change*
- `npm run start`

#### Lint
- `npm run lint`
- `npm run lint --fix`


#### Build
`npm run build`

#### Test
`npm run test`
