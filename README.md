# node-typescript
The "perfect" starter template for node + typescript project

### Usage

### Installation

```bash
# Clone the repository
gh repo clone Aayush9029/node-typescript

# Remove the .git folder
cd node-typescript && rm -rf .git
```

### npm commands
    
#### Run
- `npm run dev` *auto reloads on change*
- `npm run start`

#### Lint
- `npm run lint`
- `npm run lint --fix`


#### Build
`npm run build`

#### Test
`npm run test`
